# Tools

Calculators, spreadsheets, and job-site checklists.

## Subfolders

- **calculators/** — Spreadsheets and scripts for string sizing, conductor sizing, voltage drop, conduit fill, and production estimates.
- **checklists/** — Structured verification lists for pre-energization, commissioning, and AHJ inspection readiness.

## Available Tools

| Tool | Type | Purpose |
|---|---|---|
| String sizing worksheet | Spreadsheet | Min/max string length with temp correction |
| Conductor sizing calculator | Spreadsheet | 690.8 sizing with correction/adjustment factors |
| Pre-energization checklist | Checklist | DC and AC verification before first power-up |
| Commissioning checklist | Checklist | Inverter startup, monitoring config, production verification |
