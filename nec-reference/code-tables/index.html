NEC Code Tables

Frequently used NEC tables for conductor sizing, temperature correction, and conduit fill. These are the numbers that go into every string sizing calculation and every home-run conductor selection.

Temperature Correction Factors — Table 310.15(B)(1)

For ambient temperatures other than 30°C (86°F). Multiply the conductor's base ampacity by this factor. Applies to 60°C, 75°C, and 90°C rated conductors. PV home-run conductors in conduit on a roof see the worst of this — a 60°C correction at 50°C ambient can cut ampacity nearly in half.
