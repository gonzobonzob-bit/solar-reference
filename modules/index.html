String Sizing Quick Reference

These three calculations gate everything in NEC Article 690. Run them before anything else.

Parameter Formula NEC Reference
Max string Voc Voc × (1 + β_Voc × ΔT_cold) 690.7
Min string Vmp Vmp × (1 + β_Vmp × ΔT_hot) Inverter spec
Max string count Inverter max input current / Isc_string 690.8

Temperature Deltas

ΔT_cold = lowest expected ambient − 25°C
          Use ASHRAE 2% extreme low, or −40°F where records unavailable

ΔT_hot  = highest expected cell temp − 25°C
          Cell temp ≈ ambient + 25°C adder for roof-mounted arrays

The cold correction drives maximum voltage (690.7 compliance). The hot correction drives minimum Vmp — the array must stay above the inverter's MPPT floor on the hottest day of the year or production tanks and warranty gets complicated.
